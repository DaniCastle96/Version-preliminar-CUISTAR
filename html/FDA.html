Funcionalidad de la Aplicación

ventas

1- El sistema de información deberá permitir al administrador crear una orden de venta mediante el botón crear.

2- El sistema de información deberá permitir al cliente registrar una compra mediante el botón comprar.

3- El sistema de información deberá permitir a el adminstrador consultar una orden de venta mediante el botón buscar por medio de la barra de navegación.

4- El sistema de información deberá permitir al adminitrador descargar un reporte por medio del botón descargar reporte.

5- El sistema de información deberá permitir al administrador eliminar una orden de venta mediante el botón borrar.

Proveedores

1- El sistema de información deberá permitir al administrador realizar registros de proveedores mediante el botón agregar.

2- El sistema de información deberá permitir al administrador realizar modificaciones de registros mediante el botón actualizar.

3- El sistema de información deberá permitir al administrador visualizar los registros solicitados mediante el botón consultar datos.

4- El sistema de información deberá permitir al administrador borrar proveedores que considere innecesarios mediante el botón eliminar proveedor.

5- El sistema de información deberá permitir al administrador descargar reporte tipo Excel mediante el botón descargar.

Inventario

1- El sistema de información deberá permitir al administrador registrar productos mediante el botón Nuevo.

2- El sistema de información deberá permitir al administrador y al empleado consultar productos mediante el botón buscar en la barra de investigación.

3- El sistema de información deberá permitir al administrador actualizar productos del inventario mediante fecha, id producto,categoría,nombre de producto,,tipo de producto Unidad de Medida, Descripción de producto,Talla precio, cantidad actual y salida.

4- El sistema de información deberá permitir al administrador eliminar productos del inventario mediante dar clic en el Botón eliminar.

5- El sistema de información permitirá al administrador descargar un reporte tipo Excel donde se vea un registro del inventario mediante dar clic en el botón descargar reporte.

Fidelizacion

1- El sistema de información deberá permitir al Cliente registrar solicitud de petición, queja, reclamo o sugerencia mediante documento de identidad, nombre, tipo de PQRS, asunto.

2- El sistema de información permitirá al Cliente consultar el estado de una petición, queja o reclamo mediante dar clic en consultar solicitud.

3- El sistema de información permitirá al Cliente adjuntar archivos (jpg, png, pdf, docx) que sirvan como evidencia de su petición mediante dar clic en cargar archivo.

4- El sistema de información permitirá al cliente realizar una registro de su mascota mediante el botón registrar.

5- El sistema de información deberá permitir al Administrador atender, archivar y eliminar las peticiones, quejas, reclamos o sugerencias mediante dar clic eliminar, atender y archivar.

6- El sistema de información permitirá al Administrador realizar una consulta del listado de los PQRS y exportar los resultados a Excel mediante dar clic en exportar resultados.

7- El sistema de información permitirá al Administrador eliminar la mascota del cliente mediante el botón eliminar.

8- El sistema de información permitirá al Administrador consultar la mascota del cliente mediante el botón buscar

9- El sistema de información permitirá al Administrador actualizar la mascota del cliente mediante el botón actualizar.
